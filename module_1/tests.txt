-Пользовательский сценарий 1.1
Регистрация пользователя
--Тестовый сценарий 1.1.1
Регистрация нового пользователя
Предусловия:
Пользователь не залогинен
Шаги:
1. Открыть http://selenium1py.pythonanywhere.com/ru/accounts/login/
2. Ввести валидный адрес электронной почты в поле "Адрес электронной почты" в разделе "Зарегистрироваться", Введенный адрес отображается
3. Ввести пароль в поле "Пароль" в разделе "Зарегистрироваться", Пароль скрыт точками
4. Повторно вести пароль в поле "Повторите пароль", Пароль скрыт точками
5. Нажать кнопку "Зарегистрироваться", Отображается сообщение об успешной регистрации

-Пользовательский сценарий 1.2
Логин в личный кабинет
--Тестовый сценарий 1.2.1
Логин в личный кабинет
Предусловия:
Пользователь не залогинен
Шаги:
1. Открыть http://selenium1py.pythonanywhere.com/ru/accounts/login/
2. Ввести зарегистрированный в системе адрес электронной почты в поле "Адрес электронной почты" в разделе "Войти", Введенный адрес отображается
3. Ввести пароль в поле "Пароль" в разделе "Войти", Пароль скрыт точками
4. Нажать кнопку "Войти", Успешный логин в личный кабинет

-Пользовательский сценарий 2.2
Просмотр товара в корзине
--Тестовый сценарий 2.2.1
Переход в корзину после добавления товара
1. Открыть http://selenium1py.pythonanywhere.com/ru/catalogue/
2. Выбрать любой товар и нажать кнопку "Добавить в корзину", Отображается сообщение "Стоимость корзины теперь составляет <сумма товаров в корзине>", кнопка "Посмотреть корзину", кнопка "Оформить"
3. Нажать кнопку "Посмотреть корзину", Происходит переход на страницу корзины
